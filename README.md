# mlalgo / sinuc

Репозиторий с примерами детекции аномалий во временных рядах на синтетических синусоидальных данных.

## Описание

Проект содержит два подхода к поиску аномалий:
1. **Статистический метод** (`detect.py`) — скользящее окно с Z-score.
2. **Машинное обучение** (`detectai.py`) —Isolation Forest на инженерных признаках с автоматическим выбором порога по "колену" (knee method).

Данные генерируются классом `SineGenerator` (`gener.py`): синусоида с аддитивным шумом и случайными точечными аномалиями.

## Структура репозитория

```
sinuc/
├── gener.py           # Генератор синтетических синусоидальных рядов с аномалиями
├── create_dataset.py  # Создание датасета из 100 рядов (3000 точек каждый)
├── detect.py          # Статистический детектор (скользящее окно + Z-score)
├── detectai.py        # ML-детектор (Isolation Forest + knee threshold)
├── test.py            # Оценка статического детектора по Precision/Recall/F1
├── test_detect.py     # Оценка статического детектора с сохранением предсказаний
├── testml.py          # Оценка ML-детектора по Precision/Recall/F1
└── test_data.csv      # Сгенерированный датасет (100 рядов)
```

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Использование

### 1. Генерация данных

```bash
cd sinuc
python create_dataset.py
```

Создаст `test_data.csv` с 100 временными рядами по 3000 точек в каждом.

### 2. Запуск статистического детектора

```bash
python test.py
# или
python test_detect.py
```

### 3. Запуск ML-детектора

```bash
python testml.py
```

### 4.Запуск сайта

```bash
streamlit run test_detect.py
```


## Параметры генератора

| Параметр       | Значение | Описание                          |
|----------------|----------|-----------------------------------|
| `n_points`     | 3000     | Число точек в одном ряде          |
| `noise`        | 0.1      | Стандартное отклонение шума       |
| `anomaly_count`| 150      | Количество аномальных точек       |
| `anomaly_scale`| 2.0      | Амплитуда аномалии                |
| `amplitude`    | 1.0      | Амплитуда синусоиды               |
| `frequency`    | 1.0      | Частота синусоиды                 |
| `start`/`end`  | 0 / 64π  | Диапазон по оси x                 |

## Метрики оценки

Для каждого ряда вычисляются:
- **Precision** — доля верно найденных аномалий среди всех найденных.
- **Recall** — доля найденных аномалий среди всех реальных.
- **F1-score** — гармоническое среднее Precision и Recall.

В конце выводятся усредненные метрики по всему датасету.
